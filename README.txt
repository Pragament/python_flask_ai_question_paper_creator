Your Flask App
==============

Thank you for installing Your Flask App! This application generates question papers with prerequisite chapters.

Prerequisites
-------------
1. **Ollama**: Install Ollama from https://ollama.ai/. After installation, run:
   ollama pull llama3
   ollama pull gemma3:1b
   ollama pull gemma2:2b
2. **Internet Access**: Required for API calls to https://staticapis.pragament.com.
3. **Font**: The DejaVuSans.ttf font is included in the fonts/ directory. If missing, download it from [insert link].

Running the App
--------------
install the requirnment and run this pyinstaller YourFlaskApp.spec
1. Double-click the "Your Flask App" shortcut on your desktop or Start Menu.
2. Open a web browser and go to http://127.0.0.1:5000.
3. Follow the on-screen instructions to select chapters and generate question papers.

Troubleshooting
---------------
- If the app fails to start, ensure Ollama is installed and running.
- Ensure the app directory (C:\Program Files\YourFlaskApp) has write permissions for saving JSON files.
